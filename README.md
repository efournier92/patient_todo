# Pregnancy To Do

## Overview
- A todo-style application for tracking medical-related tasks during pregnancy.

## Versions

### Ruby
- 3.1.2

### Rails
- 7.0.3

### Vue.JS
- 3.2.33

## Database

### Version
- PostgreSQL 14

### Creation

```bash
rails db:create
```

### Initialization

```bash
rails db:migrate
```

### Populate Seed Data

```bash
rails db:seed
```

## Application

### Serve

```bash
rails s
```
